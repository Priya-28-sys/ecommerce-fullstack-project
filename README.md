# E-Commerce Full Stack Web Application

## Project Overview

A full-stack e-commerce web application built using React.js and Django REST Framework. The application supports product browsing, product details, shopping cart functionality, search filtering, responsive design, and dark mode.

## Features

* Product listing
* Product details page
* Add to cart
* Quantity management
* Search functionality
* Dark/Light mode
* Responsive mobile design
* REST API integration
* Live deployment

## Tech Stack

### Frontend

* React.js
* React Router
* Axios
* CSS

### Backend

* Django
* Django REST Framework

## Deployment

* Frontend: Vercel
* Backend: Render
